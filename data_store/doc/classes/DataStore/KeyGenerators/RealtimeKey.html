Module DataStore::KeyGenerators::RealtimeKey
In: lib/data_store/helpers.rb

This module is a realtime key generator for the key. It uses current time, thead_id along with process_id to generate a unique time-based key

Usage Example 

  class Model
    extend DataStore::KeyGenerators::RealtimeKey
  end
